Пояснение к файлу demo.png

Работа с сервером:
1. Выбор папки для синхронизации (внутри обязательно должен быть хотя бы 1 файл).
2. Нажатие кнопки "Запуск". Сервер запускается.
(При желании можно дополнительно сменить порт сервера).

Работа с клиентом:
3. Выбор папки, куда будут загружены файлы.
4. Подключение к серверу.
5. После подключение появится возможно скачать файл с сервера.
6. Можно выбрать из списка файл.
7. Начать скачивать файл.
(При желании можно дополнительно сменить адрес и порт сервера, адрес сервера localhost будет работать только если сервер запущен на том же компьютере).




Запуск программ:
1. Распаковываем архивы.
2. в InteliJ IDEA выбираем File->Open
3. Указываем путь к папке с распакованным архивом.
4. Собираем проект и запускаем на исполнение файл Runner.

Проверено на Windows XP + IntelliJ IDEA 2016.3.4
